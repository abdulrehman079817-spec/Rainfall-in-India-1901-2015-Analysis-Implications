This project looks at 115 years of historical rainfall data from 1901 to 2015. It uses Microsoft Excel to find long-term trends, regional differences, monsoon patterns, and extreme weather events across India. The focus is on Maharashtra. 
I created all dashboards, charts, and insights using Excel Pivot Tables, Pivot Charts, slicers, and complex formulas. Project Highlights - Annual rainfall trend analysis by subdivision - Identification of the wettest and driest years - Analysis of drought-prone and flood-prone years - Breakdown of seasonal and monsoon failures - In-depth look at extreme rainfall in Maharashtra Tools and Techniques Used - Microsoft Excel - Pivot Tables and Pivot Charts - Slicers for interactive filtering - Conditional Formatting - Advanced Excel formulas like SUMIFS and AVERAGEIFS - Data cleaning and aggregation This project showcases strong skills in data cleaning, analysis, dashboard design, and storytelling using Excel.
